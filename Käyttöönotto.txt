Tietokanta : Postgre

Backend : Spring Boot
    1. Luo <projektikansio> ja siirry projektikansioon
    2. Aja komento : git clone https://github.com/solita-borjeba/keskustelupalsta-harkka-borje.git
    3. Siirry IDE:een ja sulje kaikki
    4. Welcome näytöltä -> klikkaa Import project ... -> valitse Maven jne.
    5. Käynnistä backend IDE:llä


Front end : React
    1. Luo <projektikansio> ja siirry projektikansioon
    2. Aja terminaalikomento : git clone https://github.com/solita-borjeba/keskustelupalsta-frontend.git
    3. Siirry hakemistoon keskustelupalsta-frontend
    4. Aja terminaalikomento : npm ci
    5. Aja terminaalikomento : npm start

    Odotettu tulos : localhost:3000 sivu aukeaa selaimessa.
